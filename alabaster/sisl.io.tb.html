write_tb(tb, hermitian=True, **kwargs)¶

Writes the tight-binding model to the file

Writes a tight-binding model to the tight-binding file. The file can be constructed by the implict force of Hermiticity, or without.

Utilizing the Hermiticity we reduce the file-size by approximately 50%.

Parameters:

tb : TightBinding model

hermitian : boolean=True

whether the stored data is halved using the Hermitian property
