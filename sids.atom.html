class sids.atom.PeriodicTable¶

Bases: object

Basic periodic table.

It retains all periodic elements and allows easy conversion from full names to the atomic number to their weights.

radii(key, radii='calc')¶

Returns the atomic radii Return the atomic radii.

key : array_like, str, int
Uses value to lookup the atomic mass in the PeriodicTable object.
radii : str
There are 3 different radii stored:
  1. calc, the calculated
  2. empirical, the empirically found values
  3. vdw, the van der Waals found values
radii : ndarray, float
The atomic radii in Ang
class sids.atom.Atom(Z, R=-1.0, orbs=1, mass=None, tag=None)¶

Bases: object

Object to handle atomic mass, name, number of orbitals and orbital range.

The Atom object handles the atomic species with information such as - atomic number - mass - number of orbitals - radius of each orbital

Z : int
atomic number
R : ndarray
radius of orbitals belonging to the Atom
orbs : int
number of orbitals belonging to the Atom
mass : float
mass of Atom
Z : int, str
key lookup for the atomic specie, Atom[key]
R : array_like, float
the range of the atomic orbitals
orbs : int
number of orbitals attached to this atom NOTE: Length of R precedes this quantity.
mass : float, optional
the atomic mass, if not specified uses the mass from PeriodicTable
tag : str
arbitrary designation for user handling similar atoms with different settings
